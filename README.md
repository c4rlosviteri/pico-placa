# Pico y placa

Preview https://pico-placa.now.sh

## Installation

Clone repo

```sh
git clone git@github.com:c4rlosviteri/pico-placa.git
```

Install dependencies using [Yarn](https://yarnpkg.com)

```sh
yarn
```

## Dev server

```sh
yarn start
```

## Build for production

```sh
yarn build
```

## Run test with [Jest](https://jestjs.io)

```sh
yarn test
```
